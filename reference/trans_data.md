# Transect Data

Transect Data

## Usage

``` r
trans_data
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 6
rows and 2 columns.

## Examples

``` r
trans_data
#> # A tibble: 6 × 2
#>   Distance Habitat
#>      <dbl>   <dbl>

#> 2      2     10.1 
#> 3      2.5   10.2 
#> 4      3     10.2 
#> 5      4     10
```
